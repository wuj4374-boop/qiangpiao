# 图标资源

此目录包含应用程序图标。

所需文件：
- `icon.ico` - Windows图标文件
- `icon.icns` - macOS图标文件  
- `icon.png` - Linux图标文件

## 图标创建说明

### Windows (.ico)
1. 使用工具创建256x256像素的ICO文件
2. 命名为 `icon.ico`
3. 推荐工具: https://icoconvert.com/

### macOS (.icns)
1. 使用工具创建512x512像素的ICNS文件
2. 命名为 `icon.icns`
3. 推荐工具: https://cloudconvert.com/png-to-icns

### Linux (.png)
1. 创建512x512像素的PNG文件
2. 命名为 `icon.png`

## 默认图标
如果找不到图标文件，Electron将使用默认图标。

## 图标设计建议
- 尺寸: 512x512像素
- 背景: 透明
- 颜色: 科技蓝或黑色
- 样式: 简洁现代
- 包含: 门票图标或"D"字母
